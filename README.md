## Về dự án

### Trang Web này được xây dựng với các ngôn ngữ và thư viện sau:

-   HTML, CSS
-   JavaScript
-   Reactjs
-   Bootstrap
-   ClassNames
-   Tippy

### Cách sử dụng:

1. Cài đặt phiên bản React 18, npm hoặc yarn
2. Trong terminal, sử dụng câu lệnh sau để mở dự án trên web:

-   npm
    ```sh
    npm start
    ```
